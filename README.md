# Art Cottage Bakery — Website Mockup

A first-draft homepage mockup for Art Cottage Bakery, an entirely from-scratch home bakery making edible art.

Design direction: posh, dainty, feminine, artful/painterly, ethereal. Built as a single self-contained `index.html` (Cormorant Garamond + Jost + Pinyon Script, soft blush/sage/champagne-gold palette, watercolor wash, photo gallery, and an order request form). Images are placeholders, ready for the bakery's own photography.

Live preview: https://krishnatheaverage.github.io/art-cottage-bakery/

Design mockup by Krishna Harish.
